# Guia de Contribuição ✨

Ficamos felizes por você querer contribuir com o **DocGen Pro**! Este projeto visa facilitar a automação de documentos para todos.

## Como Contribuir

1.  **Reportando Bugs:** Abra uma *Issue* descrevendo o problema, como reproduzi-lo e qual comportamento era esperado.
2.  **Sugerindo Melhorias:** Se você tem uma ideia incrível, abra uma *Issue* de "Feature Request".
3.  **Enviando Pull Requests:**
    *   Faça um Fork do repositório.
    *   Crie uma branch para sua modificação (`git checkout -b feature/nova-funcao`).
    *   Faça o commit das suas alterações com mensagens claras.
    *   Envie o PR!

## Padrões de Código

- Sigam as recomendações da **PEP 8** para Python.
- Comentem partes complexas do código (preferencialmente em Português ou Inglês).
- Testem a interface gráfica (`main.py`) antes de enviar o PR.
